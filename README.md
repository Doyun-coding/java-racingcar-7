# java-racingcar-precourse


### 기능 1. 입력 처리
- ***세부 기능***
    1. 자동차의 이름을 입력 받는다.
    2. 시도 횟수를 입력 받는다.
    3. 입력 값의 유효성을 검증한다.
    4. 자동차 객체를 생성한다.

### 기능 2. 경주 진행
- ***세부 기능***
    1. 시도 횟수만큼 경주를 반복한다.
    2. 이동 여부를 결정한다.
    3. 자동차를 이동시킨다.
    4. 각 차수별 결과를 출력한다.

### 기능 3. 우승자 찾기
- ***세부 기능***
    1. 가장 많이 전진한 자동차를 찾는다.

### 기능 4. 최종 결과를 출력
- ***세부 기능***
    1. 최종 우승자를 출력한다.
